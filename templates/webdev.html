What is a website ?

A website is like a book. Just as a book consists of many pages, a website consists of many web pages, each having information, in relation to the topic/purpose of the website. Just as a book needs a publication to get published and a book seller to place it so as to make it available to the world, in the similar manner, a website needs web hosting so that it becomes available online to the users and a web server which displays a website. Just as a book has its name, by which it is known in the market, a website also has a domain name by which it is addressed.
